Software



En la carpeta Software del CD-ROM podrá encontrar un conjunto de programas, recopilados para ofrecerle
los recursos de software de los que se habla a lo largo del libro.
Si utiliza Internet Explorer en Windows, o un file manager en Linux (tales como KFM o Konqueror), le
bastará con hacer clic en los distintos enlaces que figuran a continuación para acceder al directorio donde
se encuentra el programa.

 

Apache HTTP Server: Se incluye la última versión del servidor Apache.
Apache 2.0.44 para Windows se encuentra en el directorio Software\Apache\Windows\Binary. Pulse dos veces sobre apache_2.0.44-win32-x86-no_src.msi para iniciar el instalador de Apache.
El código fuente de Apache 1.3.27 para Linux se encuentra en Software\Apache\Unix\Source.
Si dispone de tar GNU, utilice httpd-1.3.27.tar.gz. En caso contrario, utilice httpd-1.3.27.tar.Z.

MySQL: se incluye el servidor de bases de datos MySQL, tanto para Linux como para Windows.
MySQL para Windows se encuentra en el directorio Software\MySQL\Windows\Binary. Descomprima el archivo .zip y pulse dos veces sobre SETUP.EXE para iniciar el programa de instalación de MySQL.
Los instaladores binarios para MySQL 3.23.54a para Linux se encuentran en \MySQL\Unix\
Binary. Si su sistema utiliza el administrador RPM para instalar software, utilice MySQL-3.23.54a-1.i386.rpm para instalar la parte del servidor de MySQL y utilice MySQL-client-3.23.54a-1.i386.rpm para instalar la parte del cliente. Si su sistema Linux no utiliza el administrador RPM para instalar software, utilice mysql-3.23.54a-pc-linux-i686.tar.gz para instalar las partes de cliente y servidor de MySQL.
El código fuente para MySQL 3.23.54a para Unix se encuentra en Software\MySQL\Unix\
Source\mysql-3.23.54a.tar.gz. Los usuarios de Solaris deben descargar tar GNU para extraer estos archivos debido a un error en la versión de Solaris del programa tar.
Versión

Se incluye la versión 3.23.54a para Windows y Linux/Unix, así como la última versión de MySQL.

PHP: El CD-ROM incluye la distribución de la fuente de PHP.

PHP 4.3.0 para Windows se encuentra en el directorio Software\PHP\Binary. Puede utilizar el instalador simplificado (php-4.3.0-installer.exe) que le permite configurar PHP para utilizarlo con MySQL pero carece de opciones y módulos, o puede extraer el paquete completo (php-4.3.0-Win32.zip) y seguir las instrucciones detalladas en el fichero Readme del archivo Zip para configurar PHP en su sistema.
El código fuente para Linux de PHP 4.3.0 se encuentra en Software\PHP\Unix\Source\php-4.3.0.tar.
